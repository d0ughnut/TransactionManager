# TransactionManager

## Clone

```bash
$ git clone --recurse-submodules https://github.com/d0ughnut/TransactionManager.git
```

## Require Pkg

```
build-essential
cmake (>= 3.13.4)
pkg-config
libglib2.0-dev
libjsoncpp-dev
libcurl4-gnutls-dev
```

## Setup

```bash
$ cd TransactionManager
$ ./setup.sh
```

## Configure

* configure `/etc/binance/config.ini`.

## Run

```bash
$ TransactionManager
```
